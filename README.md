# 🛒 Modern E-Commerce Frontend Project

A responsive e-commerce frontend web application built using HTML, CSS, and JavaScript. This project simulates an online shopping experience with product listing, product detail view, and add-to-cart functionality.

---

## 🚀 Features
- Dynamic product listing using JavaScript
- Product detail page using localStorage
- Add to Cart functionality
- Responsive modern UI design
- Navigation bar with icons
- Product ratings and pricing display

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

## ⚙️ How to Run
1. Clone the repository  
2. Open project folder  
3. Run `index.html` in browser (or use Live Server)
